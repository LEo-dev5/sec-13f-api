ai 발전으로 쉽게 웹을 만들수있는 시대가 와서 처음으로 배포까지 완료해본 프로젝트.

 그동안 관심을 가져도 너무 이해가 안가고 힘들어서 개발을 하다가 그만두다를 5번은 반복했다.

Easy13F는 미증권거래위원회(SEC)에서 13F 공시xml 파일속에 cik로 기관명을 매칭하고 기관의 주식비중의 변화, 풋/콜옵션 구분과 Gemini api를 이용하여 전문가 페르소나를 설정해서 기관 보유량 변화 분석을 한눈에 보기 쉽게 도와주는 사이트이다.

관리자페이지에서 접속자 수 통계 SEC 데이터 수집 관리 , 유저 피드백 확인을 할 수 있다

자금이 부족해서 최대한 가성비로 해보다보니 자원이 너무 부족했고 SEC데이터를 수집할때 메모리를 과다하게 사용하게 되서 웹이 퍼지는 경우도 많았다. 

크롤링 속도를 줄이거나 기관 수를 줄여도 최적화가 되지않아서 Easy13F V2 프로젝트로 다시 만들기로 결심

Tech Stack : python Fastapi, postgresql 

Render를 이용해서 호스팅 했다.

그래도 처음으로 도메인도 구매해서 구글에 연결해서 검색했을때 뜨게도 해보고 DB에서 데이터를 조회하고 추가하고, 삭제도 해봤다. DB에 있는 데이터를 화면에 띄우는 것도 직접 해보니
